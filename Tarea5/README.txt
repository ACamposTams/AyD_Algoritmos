Para poder utilizar el proyecto se debe de abrir el archivo Tarea5.pro desde la aplicación QT.

Una vez dentro de la aplicación y con el proyecto abierto este se debe de compilar y correr.

En la siguiente ventana se muestran las diversas figuras con las que se puede trabajar, se debe de seleccionar una de ellas.

La pantalla de la aplicación tomará toda la pantalla del equipo y se mostrará la ventana donde se pueden hacer las transformaciones correspondientes según las instrucciones de cada figura.

Para salir de la pantalla completa y cambiar a otra figura sólo se debe presionar la tecla ESC.
